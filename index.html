📖 操作说明

🖱️ 基础操作

  • 添加电池：从左侧电池库拖拽到画布
  • 移动电池：鼠标拖拽电池主体
  • 连接电池：从一个端口拖拽到另一个端口
  • 断开连接：右键点击连接线 → 断开连接
  • 删除电池：右键点击电池 → 删除 / 按 Delete 键
  • 重命名电池：右键点击电池 → 重命名
  • 复制电池：右键点击电池 → 复制 / Ctrl+D
  • 框选多个：按住 Shift + 鼠标拖拽框选
  • 批量移动：框选后拖拽任意一个选中的电池

🎨 电池定制

  • 添加端口：右键 → 添加输入/输出端口
  • 设置颜色：右键 → 设置颜色
  • 自定义电池：点击左侧库的 + 按钮创建新电池模板

📐 画布控制

  • 缩放：鼠标滚轮
  • 平移：鼠标中键/右键拖拽
  • 适应画布：点击工具栏"适应画布"按钮

⌨️ 快捷键

  • Ctrl+Z - 撤销
  • Ctrl+Y / Ctrl+Shift+Z - 重做
  • Delete - 删除选中
  • Ctrl+D - 复制选中
  • Ctrl+A - 全选
  • Esc - 取消选择
  • Shift+拖拽 - 框选多个

💾 文件操作

  • 导出 JSON：保存当前工作到文件
  • 导入 JSON：从文件恢复工作
  • 清空画布：删除所有电池和连接
