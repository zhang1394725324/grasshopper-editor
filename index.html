操作说明

基础操作

  • 添加电池：从左侧电池库拖拽到画布
  • 移动电池：鼠标拖拽电池主体
  • 连接电池：从一个端口拖拽到另一个端口
  • 断开连接：右键点击连接线 → 断开连接
  • 删除电池：右键 → 删除 / 按 Delete 键

框选与多选

  • 框选多个：按住 Shift + 鼠标拖拽
  • 批量移动：框选后拖拽任意选中电池
  • 全选：Ctrl + A
  • 取消选择：Esc

电池定制

  • 重命名：右键 → 重命名
  • 添加端口：右键 → 添加输入/输出
  • 设置颜色：右键 → 设置颜色
  • 复制电池：右键 → 复制 / Ctrl+D

画布控制

  • 缩放：鼠标滚轮
  • 平移：鼠标中键/右键拖拽
  • 适应画布：点击适应按钮

快捷键

  • Ctrl+Z - 撤销
  • Ctrl+Y - 重做
  • Delete - 删除
  • Ctrl+D - 复制
  • Ctrl+A - 全选

文件操作

  • 导出 JSON：保存当前工作
  • 导入 JSON：恢复工作
  • 清空画布：删除所有内容
